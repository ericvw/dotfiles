# vim

## Installation

Installation and bootstrapping of
[Vundle](https://github.com/VundleVim/Vundle.vim) for managing ViM plugins.

```shell
cd ~/.dotfiles
stow vim
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
vim +PluginInstall +qall
```
